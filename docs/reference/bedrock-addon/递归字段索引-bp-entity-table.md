# Bedrock Add-On 递归字段索引：BP 实体 / 规则 / 表类资源

## 1. 说明

这一页把 Behavior Pack 里两条主干合并：

- 实体与生成规则：`minecraft:entity`、`minecraft:spawn_rules`
- 表类/规则类资源：`blocks`、`items`、`recipes`、`loot_tables`、`trading`、`structures` 及相邻 feature consumer
