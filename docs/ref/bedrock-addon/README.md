# Bedrock Add-On 参考

## 范围

这一组文档只回答两类基础问题：

1. Bedrock Add-On **都有哪些属性**。
2. Bedrock Add-On **都是什么结构**。

本阶段先聚焦“容器、manifest、包结构、资源族、真实样例”。

## 文档列表

- `都有哪些属性.md`：整理 addon 与 pack 级别的关键属性，重点是 `manifest.json`。
- `都是什么结构.md`：整理 `.mcaddon` / `.mcpack` / 文件夹、BP/RP、目录树与资源族。
- `来源清单.md`：列出官方文档、社区文档和真实样例仓库来源。

## 阅读顺序

1. 先看 `都是什么结构.md`，建立层级感。
2. 再看 `都有哪些属性.md`，理解各层有哪些字段。
3. 需要核对出处时，回到 `来源清单.md`。
