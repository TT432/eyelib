# Bedrock Add-On 来源清单

## 使用规则

本目录文档中的来源分成三类：

1. **官方文档**：Microsoft Learn 或官方 GitHub 样例。
2. **社区文档**：高可信度社区整理，用来补充解释和实践细节。
3. **真实样例**：真实可浏览的 addon / pack 仓库，用来做事实调研，不直接等同于规范。

## 四、重点引用建议

### 1. 回答“都有哪些属性”时优先引用

1. `O5`：manifest 属性标准。
2. `O6`：manifest 官方示例。
3. `C2`、`C3`：manifest 的实际使用解释。
4. `R3` 到 `R6`：真实项目里这些属性是如何组合出现的。

### 2. 回答“都是什么结构”时优先引用

1. `O1`、`O2`：官方工作流和包结构骨架。
2. `O8`：官方样本库里的完整目录族。
3. `C1`：`mcpack` / `mcaddon` 的容器解释。
4. `R1` 到 `R6`：真实项目的实际结构跨度。
