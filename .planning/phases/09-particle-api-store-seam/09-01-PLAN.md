Create APIs under `io.github.tt432.eyelibparticle.api` only. Per Phase 9 decisions, every ID/key is `String`-first; root/MC adapters may convert `ResourceLocation` to strings later, but the particle module API must not expose `ResourceLocation`.

Target contract shapes:
```java
public interface ParticleLookupApi<T> {
    @Nullable T get(String id);
    Map<String, T> all();
    default Collection<String> names() { return all().keySet(); }
}

public interface ParticleStore<T> extends ParticleLookupApi<T>, ParticleLifecycle {
    void put(String id, T particle);
    void replaceAll(Map<String, ? extends T> replacement);
}

@FunctionalInterface
public interface ParticleIdentifier<T> {
    String identify(T particle);
}

public final class ParticlePublisher<T> {
    public void publishParticle(T particle) { ... }
    public void replaceParticles(Iterable<? extends T> particles) { ... }
}

public record ParticleSpawnRequest(String spawnId, String particleId, Vector3f position) { ... defensive copy ... }

public interface ParticleSpawnApi {
    void spawn(ParticleSpawnRequest request);
    void remove(String spawnId);
}
```
