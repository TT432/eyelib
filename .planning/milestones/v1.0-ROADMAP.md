# Archived Roadmap: client-smoke-test v1.0

**Archived:** 2026-05-07
**Status:** Complete — all 4 phases, 10 plans, 23/23 requirements satisfied

## Overview

Build a standalone NeoForge 1.20.1 mod that automates client-side smoke testing for Minecraft mods. Tests are discovered via `@ClientSmoke` annotation using bytecode-level ASM scanning (zero class loading), executed inside an auto-created creative flat world, captured as clean screenshots on the render thread, and reported as JSON — all driven by a config-gated tick state machine.
