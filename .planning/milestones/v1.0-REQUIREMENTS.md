# Requirements (Archived): client-smoke-test v1.0

**Archived:** 2026-05-07
**Status:** All v1 requirements satisfied — 23/23 complete

## v1 Requirements

### Module Structure

- [x] **MOD-01**: 创建 `eyelib-clientsmoke-annotation` Gradle 子项目（纯 JVM，零 Minecraft 依赖），包含 `@ClientSmoke` 注解定义
- [x] **MOD-02**: 创建 `eyelib-clientsmoke` Gradle 子项目（Forge 1.20.1 + legacyForge），包含运行时引擎
- [x] **MOD-03**: Root 模块通过 `compileOnly + localRuntime` 依赖 `eyelib-clientsmoke`，开发模式下可选择性加载

### Annotation

- [x] **ANN-01**: 定义 `@ClientSmoke` 注解，RetentionPolicy.CLASS，Target TYPE，包含可选的 `description`、`priority`、`modId` 属性
- [x] **ANN-02**: 通过 `ModFileScanData` 实现字节码级注解扫描，不触发类加载
- [x] **ANN-03**: 注解扫描支持发现无来源的第三方 JAR 中标记的类

### Configuration

- [x] **CFG-01**: 通过 NeoForge `ModConfigSpec` 提供配置系统
- [x] **CFG-02**: 配置项包含：`enabled`（全局开关）、`screenshotDelay`（进入世界后延迟秒数）、`reloadStabilizeTicks`（渲染稳定 tick 数）、`exitAfterSmoke`（测试完成后自动退出）
- [x] **CFG-03**: 配置文件位于 `run/client/config/clientsmoke-common.toml`，`enabled=false` 时框架完全静默

### Engine

- [x] **ENG-01**: 通过 `@EventBusSubscriber` 订阅 Forge 1.20.1 的 `TickEvent.ClientTickEvent` (Phase.START) 驱动状态机
- [x] **ENG-02**: 状态机包含完整流程：INIT → CONFIG_LOAD → SCAN → WORLD_CREATE → WORLD_WAIT → STABILIZE → TEST_EXEC → REPOSITION → HUD_HIDE → SCREENSHOT → REPORT → EXIT
- [x] **ENG-03**: 自动创建创造模式超平坦测试世界（无需用户交互）
- [x] **ENG-04**: 多阶段就绪检查：世界已加载且 player != null → 截图延迟等待 → 渲染稳定 ticks → 测试执行准备完毕

### Screenshot Capture

- [x] **CAP-01**: 在 `RenderLevelStageEvent.AFTER_LEVEL` 阶段调用帧缓冲读取截取帧缓冲
- [x] **CAP-02**: 截图自动隐藏 HUD（F1），截后恢复
- [x] **CAP-03**: 截图输出到 `clientsmoke-reports/screenshots/`，文件命名包含测试类名和时间戳

### Test Execution

- [x] **EXEC-01**: 安全加载已扫描到的测试类（`Class.forName()`），实例化并调用执行入口（构造函数即测试）
- [x] **EXEC-02**: 测试执行异常被捕获并记录到报告中，不中断后续测试
- [x] **EXEC-03**: 按 `priority` 排序执行测试（默认值相等的测试按发现顺序）

### Auto-Exit

- [x] **EXIT-01**: 当 `exitAfterSmoke=true` 时，所有测试完成后自动退出
- [x] **EXIT-02**: 采用两阶段退出：`mc.stop()` 优雅关闭 + 60 tick 倒计时 + `Runtime.getRuntime().halt(0)` 强制退出

### Reporting

- [x] **RPT-01**: 生成 JSON 格式测试报告，包含：总测试数、通过/失败数、每个测试的类名、状态、耗时
- [x] **RPT-02**: 报告输出到 `clientsmoke-reports/report-{timestamp}.json`
